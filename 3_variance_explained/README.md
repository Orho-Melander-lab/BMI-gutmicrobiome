### Description of the scripts:
1. VE_in_all_species_by_BMI.R : estimates variance explained by BMI in all 1,280 species collectively using RDA. Same analysis was run separately in males and females on the subsetted dataset (males=1, females=2).
2. VE_in_BMI_by_species.R : estimates variance explained by BMI in each of the 1,280 species. Same analysis was run separately in males and females on the subsetted dataset (males=1, females=2).
3. VE_in_diversity_by_BMI.R: estimates variance explained in alpha diversity measures (richness, Shannon index, Inverse Simpson index) by BMI. Same analysis was run separately in males and females on the subsetted dataset (males=1, females=2).
